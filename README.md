\# AutoResearcher



A multi-agent GenAI research assistant using Retrieval-Augmented Generation (RAG) with a local Large Language Model.



---



\## Features

\- Upload multiple PDFs

\- Semantic search using vector embeddings

\- Multi-agent AI (Searcher, Critic, Writer)

\- Local LLM via Ollama (no API cost)

\- Chat-based interface

\- Evaluation with numerical metrics



---



\## Tech Stack

Python, Ollama, SentenceTransformers, Streamlit, PyMuPDF



---



\## How to Run



1\. Start Ollama:

&nbsp;  ollama run llama3



2\. Run the app:

&nbsp;  streamlit run app.py



---



\## Evaluation Metrics

\- Retrieval Accuracy

\- Chunk Relevance Score

\- Answer Correctness

\- Multi-Agent Improvement

\- Response Time

\- Topic Confusion Rate
